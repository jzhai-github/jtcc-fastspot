About This Page

The "Edit Event" page is meant to allow members with appropriate privileges to edit the selected event entry.

It makes use of the following tags:

  • Channel:Form to display the channel entry form for users to edit entries (also see Calendar Channel Form documentation).
  • {field:MY_CALENDAR_FIELD_NAME} to display a Calendar custom field using the EE Fieldtype APIâ€™s display_field method.

The global sidebar on the right makes use of the following tags:

  • Calendar:Month to display a Mini Calendar that links to days that have events.
  • Calendar:Calendars to display a list of all available Calendars.
